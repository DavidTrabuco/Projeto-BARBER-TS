🪒 Projeto – Barber Studios

Este projeto foi desenvolvido com o objetivo de simular o site de uma barbearia premium, aplicando na prática os conhecimentos adquiridos em desenvolvimento front-end moderno. A aplicação conta com uma landing page completa, seções de serviços, depoimentos, localização e formulário de inscrição.
O foco do projeto é a construção de uma aplicação Vite com TypeScript, separação de responsabilidades por componentes, e um sistema de estilos organizado via objetos TypeScript com classes Tailwind — garantindo uma experiência visual sofisticada e de alto padrão.

🛠 Tecnologias e Ferramentas Utilizadas

Vite – Bundler moderno para desenvolvimento rápido
TypeScript – Tipagem estática e maior segurança no código
Tailwind CSS v4 – Estilização utilitária com separação total do HTML
HTML5 – Estrutura semântica da aplicação
Google Fonts – Tipografia premium (Playfair Display + DM Sans)


📚 Conceitos Aplicados
Durante o desenvolvimento do projeto, foram aplicados os seguintes conceitos:

Criação e organização de pastas por componente (Navbar/, Hero/, Services/, etc.)
Separação total entre lógica, estrutura e estilos (cada componente tem index.ts + styles.ts)
Padrão de objeto de estilos TypeScript — classes Tailwind nunca escritas direto no HTML
Componentização com funções que injetam HTML no DOM
Animações CSS com @keyframes e stagger de entrada
Responsividade mobile-first com breakpoints sm, md e lg
Menu hamburguer para mobile implementado em TypeScript puro
Scroll suave entre seções via utilitário utils/scroll.ts
Uso de variáveis CSS globais (--color-*) para consistência visual
Boas práticas de desenvolvimento front-end
